# Course Templates Reference

## Alternative Session Structures

### 1. Workshop Format (3-4 hours)
- Warm-up (15 min)
- Theory Input (45 min)
- Guided Practice (60 min)
- Independent Work (90 min)
- Gallery Walk & Feedback (30 min)
- Next Steps (15 min)

### 2. Seminar Format (2 hours)
- Pre-work Review (10 min)
- Interactive Presentation (40 min)
- Small Group Discussion (30 min)
- Large Group Synthesis (30 min)
- Action Planning (10 min)

### 3. Lab Format (6 hours)
- Safety & Orientation (30 min)
- Demonstration (60 min)
- Guided Experiments (180 min)
- Data Analysis (90 min)
- Report Writing (60 min)

### 4. Flipped Classroom Format
- Pre-session materials (assigned)
- Check-in & Questions (15 min)
- Application Activities (75 min)
- Peer Teaching (45 min)
- Assessment (15 min)
