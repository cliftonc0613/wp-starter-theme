  - Show different approaches
  - Represent diverse perspectives

### Celebration Protocol

- "Look at all these different approaches!"

## TROUBLESHOOTING GUIDE

### Tech Issues
- WiFi down → Use hotspot, have offline materials
- Software not working → Have browser alternatives
- Projector issues → Prepare for screen sharing

### Time Management
- Running behind → Cut 1 activity, keep showcase
- Running ahead → Add reflection questions
- Slow start → Have warm-up questions ready

### Student Engagement
- Quiet students → Call on them gently, pair work
- Dominant students –→ Redirect questions, time limits
- Confusion → Stop, clarify with examples

### Content Issues
- Too advanced → Simplify language, more examples
- Too basic → Add challenge options
- Not connecting –→ More relevant stories
