## RED FLAGS CHECKLIST

### ⚠️ Tech Issues
- WiFi down → Hotspot ready?
- Software crashing → Browser alternative?
- Display issues → Screen sharing?

### ⚠️ Engagement Issues
- Quiet room → Time for pair work?
- Confusion → Quick demo?
- Rushing → "Slow down, we have time"

### ⚠️ Time Issues
- Behind schedule → Cut which activity?
- Ahead of schedule → What extension?
- 5-minute warning → Give it!
